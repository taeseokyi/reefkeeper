# reference-aquawiz-overseas-cases

> "해외 상용 AquaWiz 커뮤니티 오차 분석(reef2reef 등) — 방CO₂·기포 오차구조가 우리 진단과 일치, 제조사 공식 해법=외기라인=우리 레버(a), ±0.5dKH 운용범위 일치"

---

해외(reef2reef/UltimateReef/BAR) 상용 AquaWiz 사용자 오차 분석 (2026-06-18 검색). **우리가 독립적으로 도달한 오차 구조와 그대로 겹침 = 진단 경로 외부 검증.**

1. **최다 오차 = 방 CO₂ 변동(우리 새벽 문제 동일).** 사용자: 방 CO₂ 불안정→매 측정 튐, "밀폐 통에 넣으니 멈춤". **★AquaWiz 공식 응답: "급격 CO₂ 변화 시 ~1시간만 영향, 흡입 튜브를 스키머 실외 공기라인에 연결하면 됨."** ⟹ "~1시간"=우리 17:08 드리프트, "외기 라인"=우리 [project-kh-v4-firmware-state](project-kh-v4-firmware-state.md) 레버 (a) 가스기준 고정 그 자체(제조사 공식 해법=우리 해법).
2. **2번째 = 폭기/기포 품질(우리 토출량 레버 동일).** 튜브 이물질/염 침착→기포 작아짐→2 dKH 이상 변동, 6개월마다 에어 청소. "기포 부족=큰 오차" 상용 실측.
3. **현장 정확도:** 공칭 ±0.05(과장 평), 실제 Hanna 대비 ≤0.2, 무보정 7주 ≤0.75 dKH. **AquaWiz 팀: "0.5 dKH 이하 변동=excellent, 추세 추적용."** ⟹ 우리 목표 ±0.5/±0.25=[project-accuracy-target](project-accuracy-target.md)가 제조사 권장범위와 일치. 보정주기 ref↔tank ΔKH 연동(1.0=주간/0.5=월간).
4. **함의:** 상용은 영점/보정·청소로 관리, **완전평형(V4)까진 안 감** → 우리 V4(진짜 평탄까지)는 한 발 더. 단 상용 운용지혜(외기라인·정기 에어청소·추세 위주)는 차용 가치.
