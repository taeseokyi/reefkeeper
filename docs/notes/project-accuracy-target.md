# project-accuracy-target

> KH 측정 목표 오차 = ±0.5 dKH 허용 / ±0.25 dKH 희망 (사용자 2026-06-18). 6/27 다일 널(n=13): 편향 −0.13 실재(σ0.061), 온도·KCl 원인 아님=액간전위. 목표는 여전히 충족

---

사용자 목표 오차(2026-06-18): **±0.5 dKH 이내면 허용, ±0.25 dKH면 희망.** 완전한 정확성은 기대하지 않음.

**★★★갱신(2026-06-27, 다일 동일시료 완전 널 캠페인 n=13, 6/23 05:00~6/27 05:00): 편향 −0.13 dKH가 실재함이 통계 확정 — 6/21 "−0.06·영점불요" 결론을 갱신.** 6/22 22:00 시료 신선화 후 장치·시료 무변경 4일·1일3회 연속 널. 참값=앵커 8.448. 결과: **편향 −0.130 dKH(95% CI [−0.167,−0.093] → 0 배제=실재), 정밀도 σ 0.061, RMSE 0.142, ΔpH(tank−ref) −6.7mpH(SD3.1, 계통).** 단발 1회 95% 예측구간 [−0.25,−0.01]=아무 1회나 ±0.25 안. **★온도·KCl은 원인 아님(사용자 결론, 편향 vs 수온 r=−0.10≈0으로 데이터 정합).** 남은 유력 원인=액간전위/기준전극 비대칭(일관된 ref>tank baseline, [project-kh-offset-aeration-test](project-kh-offset-aeration-test.md) 정적 baseline 항). **함의: ① "−0.06=영구 불요"(아래 6/21)는 −0.13으로 약 2배 커져 갱신됨 — 잔여 baseline이 더 큼. ② 그래도 RMSE 0.14·단발 ±0.25 안이라 목표는 여전히 충족(영점보정 시 −0.13 제거→±0.06 가능하나 운영상 선택). ③ 신뢰수준 측면: 널이라 σ 기지→95%CI 마진 ±0.25 n≥1·±0.1 n≥2·±0.05 n≥6, 현재 n=13으로 편향 ±0.037까지 좁혀져 추가측정 불필요.** 기록=[project-measure-kh-script](project-measure-kh-script.md) 대장 "다일 동일시료 완전 널 캠페인 통계" 섹션.

**★단발 정밀도 = 전극 물리바닥 도달, 추가 최적화 종료(2026-06-26).** 일정한 시료에서 정시 측정이 8.23↔8.41(±0.1 dKH)로 산포 → 시료 변동 아니라 **순수 계측 잡음.** dKH는 ΔpH에만 의존(감도 ~0.02 dKH/mpH)하므로 이는 ΔpH가 −0.011~−0.002로 ~9 mpH 흔들린 결과. 절대 pH는 런간 ~25 mpH 떠다니는데 차동식이 대부분 상쇄(25→9 mpH)함=정상 작동. 남은 잔차의 정체=**유리전극 드리프트(분당 1–3 mpH)+KCl 액간전위+tank→ref 26분 시간비대칭 드리프트(순서 고정이라 ΔpH에서 안 지워짐)+평탄 종점잔차(net8=1 잔여 크리프)**=이 센서 클래스의 물리/전기화학 바닥. ⟹ tank→ref 대칭화·종점강화로 한두 mpH 더 깎아봐야 ±0.02 dKH라 회귀위험 대비 무가치 → **단발 최적화 종료가 맞음.** ★중요 구분: 센서 물리한계 ≠ 산출물 한계. 산출물은 dKH 추세이고 시료가 일정하니 잔차의 랜덤성분은 N회 평균 σ/√N로 축소(계통 bias는 영점보정으로 제거). ⟹ 만약 ±0.1이 거슬리면 레버는 **통계(샘플수·이동평균·스무딩)이지 물리(더 좋은 전극) 아님**, 그것도 필요해지면 그때. 현재 ±0.1은 ±0.25 안·외부검증 "0.5 이하=excellent·추세용" 밴드 안([reference-aquawiz-overseas-cases](reference-aquawiz-overseas-cases.md))이라 멈출 자리. (관련 잔차 메커니즘=[project-kh-offset-aeration-test](project-kh-offset-aeration-test.md) "시간상수 비대칭+정적 baseline")

**★★★최종 확정(2026-06-21, 완전 널): 목표 달성, 영점보정 불요.** 위즈(5L) 물을 tank 외부챔버 100mL에도 넣어 **양쪽 같은 물=물차이 구성상 0**으로 만든 완전 널(둘 다 진짜 평탄 21/18회 O, T 27.4 안정): **ΔpH −0.003·dKH 8.387·무보정 오차 −0.061.** 물차이=0이라 이 −0.061이 *순수 계측 offset의 진짜 값*(ΔpH로 −3mpH=분해능/전극 미세 비대칭). ⟹ **계측 offset −0.06은 ±0.25의 1/4=보정 가치 없음 → 영점보정 영구 불요.** 정밀도 SD ±0.08 + offset −0.06 ⟹ **희망 ±0.25 달성 확정.** 부분 널(6/21 13:00 신선tank vs 옛5L)에서 옛 tank −0.05·오차−0.9 → 신선 −0.008·오차−0.15로 회복 → **−0.92 절대오차의 −0.86은 옛 200mL tank 물 degrade(석출), 계측 아님**이 규명됨. 실제 리프수조 정상 물이면 무보정 ±0.1 안. ([project-kh-v4-firmware-state](project-kh-v4-firmware-state.md))

**★갱신(2026-06-19, 밀폐 후): 영점보정이 "필수"에서 "선택·여유"로 완화.** 밀폐 공통 헤드스페이스 적용 후 V4 원시오차가 **−0.78 → ~−0.18(−0.165/−0.193, 05/13시)** 로 급감 → **무보정으로도 ±0.25 희망범위 안.** 과거 −0.78의 대부분은 systematic device offset이 아니라 증발+불완전평형+CO₂ 아티팩트였고 밀폐가 걷어냄([project-kh-v4-firmware-state](project-kh-v4-firmware-state.md) 밀폐 검증 ①②). ⟹ 영점보정은 이제 잔차를 0 근처로 다듬는 *옵션*이지 합격 필수조건 아님. 21:00·시계열로 offset 상수성 확인 후 결정(급하지 않음).

**(이전, 개방 셋업 기준) 영점보정 필수 논리:**
- V4 개방 널테스트 원시값 7.67/7.66/7.95 dKH vs 진짜 8.448 → 오차 −0.78/−0.79/−0.50, **무보정이면 ±0.5도 못 맞춤.** 오차의 대부분이 공통 systematic offset(~−0.75 dKH). ※밀폐로 이 전제가 바뀜(위 갱신).
- **흩어짐은 이미 거의 합격.** 세 점 흩어짐 0.29 dKH(≈±0.15). 평균 영점보정 후 잔차 −0.09/−0.10/+0.19 → **최대 0.19, ±0.25 희망치 안.**
- **"−0.042가 완벽한 이전가능 상수인가"는 과한 질문이 됨.** ±0.001 정밀도면 0.29 흔들림이 치명적이나, ±0.25 기준에선 **오프셋이 ±0.25 안으로 안정하면 충분.** [project-kh-v4-firmware-state](project-kh-v4-firmware-state.md)의 영점보류는 "정밀 상수 확인"용이었는데, 이 tolerance에선 완화 — 새 널+시계열이 "영점 뺀 잔차 ±0.25 군집"이면 합격(정확히 같은 값 불요).
- **13:00 +0.29는 시험대(개방200mL 5일 증발) 산물 가능성 — 실제 대용량+보충수 리프수조는 더 타이트.** ⟹ 실사용 흩어짐이 stress-null보다 작을 수 있어 목표 더 쉬움.
- **스크러버/미세튜닝 불필요.** 큰 사고(17:08 측정중 드리프트 산모양)만 막으면 됨. 마지막 몇 mpH 추격 금지.

판정 기준: 새 널테스트+정시 시계열에서 **영점보정 후 잔차가 ±0.25 dKH 안에 군집**(낮은 SD)이면 목표 달성. 최소 ±0.5는 영점보정만으로도 사실상 확보. [feedback-validation-independent-replication](feedback-validation-independent-replication.md)
