# project-dkh-temp-column-is-sealed-box

> "dkh.dat 온도 컬럼=밀폐 측정챔버 내부 수온(참조수·수조수 평형온도), 실제 메인 리프수조 수온의 대리값 아님"

---

`dkh.dat`/reefCore에 기록되는 온도 값은 **밀폐 측정챔버 안에서 참조수·수조수·KCl이 평형을 이룬 온도**다. 리프위즈 박스가 완전 밀폐형이라 실온보다 승온되는 게 정상(사용자 확인 2026-07-01).

**Why:** [project-kh-v4-firmware-state](project-kh-v4-firmware-state.md)의 밀폐 공통 헤드스페이스 설계 의도가 "참조수·수조수·KCl을 동일 온도로 유지→Nernst 보정 정확도 향상"이라, 밀폐로 인한 승온은 설계상 자연스러운 부작용이며 오히려 온도 균일성(정확도)엔 도움. 차동식(ΔpH만 의존)이라 ref·tank가 같은 온도를 공유하는 한 dKH 계산 신뢰도엔 영향 없음.

**How to apply:** 측정 결과 리뷰 시 dkh.dat 온도 컬럼(28~30°C대)을 보고 "메인 수조가 너무 뜨겁다"고 건강 이슈로 플래그하지 말 것 — 이건 밀폐 챔버 내부 온도일 뿐, 실제 메인 리프탱크 수온의 대리값이 아니다. 사용자가 실제 수조 수온 관리를 원하면 별도 온도계/컨트롤러 데이터를 봐야 한다고 안내.
