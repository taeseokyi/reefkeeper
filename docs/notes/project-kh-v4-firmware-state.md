**측정 코드 = V4 (`bin/measure_kh_once.py`, Windows `C:\dkh\work`에도 배포):**
- **폭기 중 측정 + 진짜 평형(평탄)까지.** 평탄 판정 = **정수 milli-pH 윈도우 span**: 최근 4개 `round(ph*1000)`의 max−min ≤ **2 mpH**. (float 비교 `abs(d)<=0.001`은 0.001 경계 지터로 못 씀 → 정수 비교. 연속-Δ는 느린 크리프에 조기 latch라 폐기.)
- 2-phase: **[A]** tank·ref 동시폭기 + tank 평탄까지 측정(ref는 홀딩서 co-aeration) → **전이 무대기**(airoff → `m1b:82` 완전배출[오염방지, 단축불가] → `m2f:60`) → **[B]** ref 평탄까지(co-aeration이라 빨리). 간격 30s.
- **무한대기 방지:** `PHASE_MAX_SECS=2400`/phase + `MEAS_MAX=80` + 연속 측정실패 `FAIL_MAX=5` + 시리얼 read 타임아웃. 미평탄 상한 도달 시 마지막값+경고(행 안 함).
- 오류/비정상 종료 시 `_safe_cleanup`(에어OFF+측정챔버 배출+KCl 소크 복원). KCl 소크 완료 미검증이면 에러로 0.0 기록.

**dkh.dat 규약** (`HH ref_pH tank_pH ref_kh(8.448) tank_kh temp`):

- **평탄 미도달 = `tank_kh` 음수(−)** 표식(값 크기 유지). 전부-0 에러표식과 구분돼 래치 안 걸림.
